Cómo Comprar

¿Cómo puedo realizar un pedido?

  1. Una vez elegido el producto, presioná el botón AGREGAR AL CARRITO
  2. Verás en tu carrito los artículos que tenés cargados
  3. Podés modificar la cantidad o eliminar los que desees
  4. En el carrito presiona el boton FINALIZAR PEDIDO
  5. Para continuar con la confirmación del pedido, deberás completar con los datos requeridos
  6. Al seleccionar "Confirmar Pedido" se registra el pedido y el equipo de sublisa te contactara para acordar forma de pago y entrega

A las 48 hs de haber realizado el pedido, si no se abona, el sistema cancela automáticamente la orden.
